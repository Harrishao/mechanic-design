# 15吨压片机 — 课程设计任务规划

> 工件：圆形片坯 φ34×5mm | 冲压力150kN | 产量25片/min | 电机2.2kW/940rpm

---

## 阶段一：方案设计

- [ ] 1.1 绘制执行机构动作分解图（上冲头、下冲头、粉筛三机构的时序配合）
- [ ] 1.2 绘制机构运动循环图（以主轴转角为横轴，标注各机构行程-时间关系）
- [ ] 1.3 确定上冲头方案：曲柄滑块机构，计算曲柄长度（行程90~100mm → 曲柄约45~50mm）
- [ ] 1.4 确定下冲头方案：凸轮机构，按位移要求（3→8→16→21mm）设计推程/回程运动规律
- [ ] 1.5 确定粉筛方案：凸轮机构，位移要求（往复振动 + 水平右移45~50mm推片）
- [ ] 1.6 拟定传动系统方案：电机→减速器→分配轴→各凸轮/曲柄，分配传动比
- [ ] 1.7 计算总传动比（940rpm → 25rpm 主轴，总减速比 ≈37.6），分配各级传动比
- [ ] 1.8 绘制整机机构运动简图（方案草图，不含精确尺寸）

---

## 阶段二：运动尺寸设计

- [ ] 2.1 上冲头曲柄滑块机构：按行程90~100mm计算曲柄、连杆长度，校验压力角
- [ ] 2.2 下冲头凸轮机构：按位移线图（0→-3→+8停→+16停→-21）设计凸轮廓线
- [ ] 2.3 粉筛凸轮机构：按水平位移要求设计凸轮廓线
- [ ] 2.4 减速器设计：确定齿轮级数、齿数、模数，计算各级齿轮几何尺寸
- [ ] 2.5 带轮传动设计：确定带型、带轮直径、中心距

---

## 阶段三：运动分析（Matlab）

- [ ] 3.1 选定分析对象（建议：上冲头曲柄滑块机构，运动明确，解析法方便）
- [ ] 3.2 建立曲柄滑块的运动学数学模型（位移/速度/加速度与曲柄转角关系）
- [ ] 3.3 Matlab编程求解，绘制滑块位移、速度、加速度曲线
- [ ] 3.4 若选凸轮机构：建立凸轮从动件运动规律，绘制位移/速度/加速度曲线
- [ ] 3.5 整理运动分析结果，备用于说明书

---

## 阶段四：三维建模与仿真（SolidWorks）

- [ ] 4.1 建立各零件三维模型（机架、曲柄、连杆、滑块、凸轮、齿轮、带轮等）
- [ ] 4.2 装配各子机构（上冲头组件、下冲头组件、粉筛组件、传动组件）
- [ ] 4.3 整机装配，检查干涉
- [ ] 4.4 设置运动仿真：定义驱动、配合关系，运行Motion分析
- [ ] 4.5 录制仿真动画视频（展示一个完整工作循环）
- [ ] 4.6 导出零部件工程图用于CAD

---

## 阶段五：图纸绘制（AutoCAD）

- [ ] 5.1 准备2号图纸（420×594mm）模板，含标题栏
- [ ] 5.2 绘制机构运动简图（最终方案，标注各构件尺寸及运动副）
- [ ] 5.3 标注关键尺寸、配合关系、图例说明
- [ ] 5.4 检查图纸格式符合机械制图标准
