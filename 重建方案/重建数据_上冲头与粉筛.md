- 凸轮回转中心 O1 = (0,0)，主轴沿 Z，**逆时针**回转（已从 下冲头凸轮_SW.txt 反推确认）

## 1. 重建要点（相对旧方案的改变）

1. **上冲头改为二级摆杆增力布置**（按手绘方案页面下方画法）：凸轮 → 下摆杆（O2）→ 长连杆 → 上摆杆（O3，直杆杠杆，臂比 70:140 = 1:2）→ 小连杆 → 冲头。摆杆需求摆角由 30° 降到 18.2°，凸轮理论基圆由 **117 → 54**，实际廓线最大半径仅 68.9；冲头长度由 231 → **130**。
2. **粉筛同拓扑缩小**：中心距 170 → **109.7**，理论基圆 113.5 → **81**，实际廓线最大半径 89.1（旧方案仅基圆就 113.5）；连杆 250 → 211.7。托架、筛口、耳板销位**不变**（销 Y=223，行程仍 48，装料位销 X=+42、筛口中心对正 X=0）。
3. **粉筛左退窗口由 90~120° 放宽到 90~135°**，以压低左退压力角。校核依据：上冲头尖端在 φ≈170° 才降到托架顶面高度（Y≈230），φ=135 退净仍有 35° 裕量；其余相位不动。⚠ 需同步更新 运动循环图 与运动方案文档中粉筛一行。
4. 两机构全部杆系位于 XY 平面（Z≈0 薄层），仅滚子沿 Z 悬臂伸至各自凸轮站位（沿用现有"z轴避让"做法）；**粉筛连杆与托架耳板置于 Z≈+45 平面**（托架前缘侧），与冲头错开。
5. 旧 整机装配参数填写表.md 中上冲头、粉筛两节作废，以本文档为准。

## 5. 凸轮廓线文件与安装相位

1. `上冲头凸轮v2_SW.txt`、`粉筛凸轮v2_SW.txt` 为**实际廓线**（已扣滚子半径），格式与 下冲头凸轮_SW.txt 相同（x TAB y TAB 0，闭合，720 点），SolidWorks「通过 XYZ 点的曲线」直接导入后拉伸。
2. 廓线在**真实机构坐标系**中生成：文件姿态即装填位（φ=0）姿态。装配时把主轴置于装填位（下冲头装填位、上冲头最高、粉筛在装料位），三凸轮**按文件原始姿态装上即可，无需再旋转**（即"装填位对齐法"，旧表中 0°/128°/201° 的数值安装角作废）。
3. 校对基准：装填位时上冲头凸轮滚子中心应在方位角 −33.5°、半径 54 处；粉筛凸轮滚子中心在方位角 −128.7°、半径 99.07 处（粉筛装填位为全升程）。
4. `.csv` 含 φ、摆角 ψ、节曲线半径、实际廓线半径、压力角，可用于说明书运动分析章节。

## 6. 建模与装配顺序建议

1. 按 §3.2 / §4.2 建摆杆、连杆、上摆杆（直杆）、缩短后的上冲头；凸轮由 txt 导入拉伸（宽 30 / 20，孔 φ20 + 键槽）。
2. 摆杆滚子臂照旧做 z 向避让：滚子悬臂伸至各自凸轮中面（上冲头 Z=67.5、粉筛 Z=132.5），杆系本体留在 Z≈0 薄层；粉筛连杆与托架耳板在 Z≈+45 平面。
3. 机架需新增三个固定支点：O2(100,35)、O3(140,457.04)、O2s(−100,−45)，及 X=0、Y≈308~358 的冲头导向套。
4. 台面右缘控制在 X ≤ 180（上冲头长连杆运动区 X≈206~248）；左缘照旧给托架让位。
5. 铰点销径沿用现有 d8 / d20 销件体系。
6. 上冲头、粉筛各加一根复位拉簧（力封闭），分别挂在 E/D 附近与摆杆连杆臂上。
