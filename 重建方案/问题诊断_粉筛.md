# 粉筛两问题诊断（对应 待办事项2.md）

## 结论先行

设计计算上**不存在**行程不足：销行程精确 48.000，左停位托架右缘 X=−20 与上冲头 ±17 留有 3mm 间隙，装料位筛口正对型腔。托架本体建模没有错（销-左端面 69 与图纸一致）。问题出在**滚子没有被约束在凸轮上**（或装配中粉筛凸轮不是 v2 件）。

## 定量证据

把实测左端面位置 +69 换算成托架销位置，再按设计杆系反解滚子中心位置：

| 状态 | 销 X (实测换算) | 反解滚子中心距主轴 | v2 凸轮可达范围 |
|---|---|---|---|
| 实测右止点 | +73.48 | **118.16** | 81 ~ 99.07 → 超出 19mm，滚子悬空 |
| 实测左止点 | +23.06 | **91.09** | 应为 81（基圆）→ 差 10mm |
| 设计装料位 | +42 | 99.07（全升程） | ✓ |
| 设计左停位 | −6 | 81.00（基圆） | ✓ |

实测的两个"止点"对应的滚子位置都不在凸轮廓线上——右止点时滚子离凸轮表面悬空约 19mm。也就是说手动拖动时整套杆系在自由摆动，"止点"是别的约束（如拖动极限）给出的，并非凸轮给定的行程。这同时解释了行程 50.4≠48。

## 修复步骤（SolidWorks）

1. 确认装配中的粉筛凸轮是 v2 件：量实际廓线最小/最大半径应为 71.0 / 89.07（若量到 103.5 左右则是旧 v1 凸轮，替换之）。
2. 给滚子外圆面与凸轮廓线面添加**机械配合 → 凸轮配合（相切）**，确保全程贴合；检查该配合未被压缩。
3. 静态校核两位置（转动主轴）：左停位滚子中心距主轴 81.00、销 X=−6；装料位 99.07、销 X=+42。
4. 相位按基准面对齐法复位（见前述：凸轮零件右视基准面与装配体右视基准面成 0°，粉筛远休弧中点应指向 198.8°）。
5. 若做 Motion 仿真且用"实体接触"替代凸轮配合，必须加复位弹簧（力封闭），否则左退段滚子脱离廓线，会复现同样的漂移。
6. 若完成 1~2 后位置仍不对，依次核对：O2s=(−100,−45)、Rb=110、Lr=50、两臂夹角 129.77°、连杆孔距 211.71、销高 Y=223、滚子 d20。

## 关于"运动周期不符合预期"

这不是相位或机构错误，而是上述偏移造成的错觉，叠加方向约定的混淆：

- v2 设计的循环是你所述"正确流程"的**左右镜像**，功能完全等价：**右移一个冲程同时完成推卸与装料**——前缘把型腔顶面的成型片向右推走，紧随其后的筛口落到型腔上方装料（φ 290~335°）；随后 335~90° 停在右侧筛料装料，90~135° 左退避让，135~290° 冲头压片与顶出。
- 现在观察到"左移=装料"，是因为整个行程带右偏了约 30：右止点筛口中心在 +31.5（没对上型腔），左止点筛口中心在 −19（反而凑近型腔），看起来像左移装料；压片时托架又停在型腔上方，于是干涉。
- 把滚子-凸轮约束修好、位置回到 −6~+42 后，循环自然恢复为：右移(推卸+装料) → 右停(装料) → 左退 → 压片/顶出 → 右移，与运动循环图一致，无需改任何零件或凸轮。
