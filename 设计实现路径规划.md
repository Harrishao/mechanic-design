# 15吨压片机 — 设计实现路径规划

> 工件 φ34×5mm 圆片 | 冲压力 150kN | 产量 25 片/min | 电机 2.2kW、940 r/min
> 三执行机构：上冲头、下冲头、粉筛，单电机驱动

按机械设计常规流程划分为六个 MVP，每个阶段产出一个可独立评审或直接使用的成果，前一阶段是后一阶段的输入。

---

## MVP 1 · 运动方案

产出：一份可评审的整机运动方案（循环图 + 机构选型 + 传动方案 + 运动简图草图）

- [ ] 1.1 拟定上冲头、下冲头、粉筛三构件在一个工作周期内的动作时序
- [ ] 1.2 绘制机构运动循环图（主轴转角 0~360° 为横轴，标注各机构行程区间与停歇段）
- [ ] 1.3 选定上冲头机构型式（行程 90~100mm，下压并保压）
- [ ] 1.4 选定下冲头机构型式（位移规律 −3 → +8 停 → +16 停 → −21mm）
- [ ] 1.5 选定粉筛机构型式（往复筛料 + 右移 45~50mm 卸料）
- [ ] 1.6 拟定电机→主轴→各执行机构的传动系统方案
- [ ] 1.7 计算总传动比（940 → 25 r/min，i ≈ 37.6）并分配各级传动比
- [ ] 1.8 绘制整机机构运动简图（方案级，不含精确尺寸）

---

## MVP 2 · 运动尺寸设计

产出：全部机构与传动件的具体几何参数，达到可建模精度

- [ ] 2.1 上冲头机构尺寸综合，校核压力角 / 传动角
- [ ] 2.2 下冲头机构尺寸综合（凸轮则按位移规律设计廓线）
- [ ] 2.3 粉筛机构尺寸综合
- [ ] 2.4 带传动设计：带型、大小带轮直径、中心距
- [ ] 2.5 减速齿轮设计：齿数、模数、中心距、几何尺寸
- [ ] 2.6 汇总全部尺寸，形成几何参数表

---

## MVP 3 · 运动分析

产出：一个执行机构的解析运动学结果与曲线

- [ ] 3.1 选定分析对象（建议上冲头机构，运动明确、解析方便）
- [ ] 3.2 建立位移、速度、加速度关于主轴转角的解析模型
- [ ] 3.3 Matlab 求解并绘制位移 / 速度 / 加速度曲线
